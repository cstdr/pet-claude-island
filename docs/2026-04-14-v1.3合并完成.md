## 核心改动

### 1. 安全修复
- Socket 权限: `0o777` → `0o600` (防止任意进程伪造 hook 事件)

### 2. Task → Agent 兼容
- 工具名判断统一使用 `ToolCallItem.isSubagentContainerName()`
- 支持 `Agent` 作为 `Task` 的别名

### 3. 新 Hook 事件
- `PostToolUseFailure`
- `PermissionDenied`
- `StopFailure`
- `SubagentStart`
- `PostCompact`

### 4. 本地化
- `LanguageManager` 支持运行时切换语言
- `.localized` 字符串扩展
- 中英文双语支持

### 5. ClaudeDirPickerRow
- 原生 macOS 文件夹选择器
- 自动检测 / 手动选择两种模式
- 展开时菜单高度自动调整

### 6. 菜单高度自适应
- `GeometryReader` + `PreferenceKey` 测量实际内容高度
- 任意 picker 展开时自动调整窗口大小

### 7. 其他改进
- Dead-process 检测 (~3秒定期检查)
- `ImageMessageView` 内联图片渲染
- `UsageInfo` token 追踪
- `StableHash` 稳定 ID 生成
